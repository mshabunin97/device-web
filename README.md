# [device-website](https://mshabunin97.github.io/device/)
Этот многостраничный сайт разработан для предоставления услуг по аренде электросамокатов, велосипедов и электровелосипедов. Он включает в себя различные функциональные возможности, которые обеспечивают удобство использования для пользователей.
